  timestamps/durations.

  high-contrast = brutalism.

  All real logic is JS. Zero type coverage where it matters (data layer, hooks,
  components).

  Key fact: tsconfig already exists, strict: true, typescript + @types 
  installed. TS is set up and unused. .jsx files compile today only because
  expo/tsconfig.base allows JS (allowJs). Expo/Metro (Babel) strips types ‚ÄĒ no
